			- list테이블 : 회원 아이디, 제목, 작성일, 내용,
						영화제목, 영화코드(영진원), 영화코드(네이버)
						 글번호, 게시판 코드, 공개여부, 추천, 비추천, 신고, 조회수

				- 1 : 공개여부
				- 2 : 게시판 코드
				- sysdate : 작성일
				- 영화제목은 |로 연결
				- 글번호는 시퀀스 사용 
				- 회원 아이디는 세션에서 가져올 것

			- log테이블 : 활동시간, 활동분류, 활동ID, 제목, 글번호, 작성자ID
				- 활동분류 : 1
				- 활동시간, 글번호 : list테이블에 insert된 데이터와 동일해야 함
				- 활동ID, 작성자 ID : 세션에서 가져올 것
				- 제목 :
